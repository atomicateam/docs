atomica.defaults.make_demo_scenarios(proj)[source]¶

Create demo scenarios

This method creates three default budget scenarios

  • Default budget
  • Doubled budget
  • Zero budget

The scenarios will be created and added to the project’s list of scenarios

Parameters:dorun – If True, and if doadd=True, simulations will be run
Return type:None
